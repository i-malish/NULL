Web-додаток для швидкого і зрозумілого пошуку вакансій за заданими параметрами, а такoж зручного розміщення вакансій роботи.
-	мова програмування: Ruby, Ruby on Rails;
-	База даних: PostgreSQL;
-	У	 додатку є три основні ролі:
а) незареєстрований користувач – роль в якій буде відбуватися пошук вакансій (роботи).
б) зареєстрований користувач – роль в якій роботодавець буде створювати компанії і відповідні вакансії до компаній.
в) адміністратор додатку, для цього використано Active Admin
Active Admin is a Ruby on Rails plugin for generating administration style interfaces. It abstracts common business application patterns to make it simple for developers to implement beautiful and elegant interfaces with very little effort.  
Методи і технології: Ruby, Ruby on Rails, RubyMine, PostgreSQL, Twitter Bootstrap, jQuery
